-- 预提结果主表

```sql
select * from T_FULE_INVOICE_RECEIPT_COSTS ORDER BY timestamp desc
select * from T_FULE_INVOICE_RECEIPT_COSTS where to_char(timestamp)='1779751801';
-- 检查预提结果：看三张表有没有⾦额为空的数据，如果有，检查相应站点的油价
-- 预提结果表（与油单表匹配）
SELECT * FROM T_FUEL_INVOICE_RECEIPT_DETAIL d WHERE  timestamp='1779751801' and d.fuel_amount is
null;
-- 预提结果表（与新加油表匹配）
SELECT * FROM T_FLIGHT_FUELINFO_DETAIL d WHERE  timestamp='1779751801' and d.amount is null;
-- 预提结果表（根据机型⻜⾏时间估算）
SELECT *  FROM T_TYPE_LINE_DETAIL d WHERE timestamp='1779751801' and d.amount is null;
-- 预提结果表（传共享，按公司、机型、航线、币种维度）
select * from aop_dcs_ap_request WHERE timestamp='1779751801';
```

预提存储过程：
PKG_FUEL_FEE_ACCRUED.FUEL_FEE_MAIN_JOB  每⽉ 1 号定时任务⼊⼝
PKG_FUEL_FEE_ACCRUED.FUEL_FEE_MAIN  将航班信息根据固定规则移⼊预提结果表、三张明细表
PKG_FUEL_FEE_ACCRUED.FUEL_FEE_CALCULATION   T_FUEL_INVOICE_RECEIPT_DETAIL 表中⾦额计算（与油
单表匹配部分）
PKG_FUEL_FEE_ACCRUED.FUEL_FEE_EXPRESS    T_FLIGHT_FUELINFO_DETAIL 表中⾦额计算（与新加油表匹配
部分）
PKG_FUEL_FEE_ACCRUED.FUEL_FEE_OTHER      T_TYPE_LINE_DETAIL 表中⾦额计算（根据机型⻜⾏时间估算部
分）
PKG_FUEL_FEE_ACCRUED.FUEL_FEE_FOR_SOA   共享预提结果表 aop_dcs_ap_request 数据插⼊
-- 对于 amount 为空的这些航班，要根据始发站去系统⾥查询，是否有做油价录⼊（国内油价管理、国际油价管理），
补录油价之后  ，可以重新计算这些航班的⾦额。（重新计算预提⾦额时，要确保共享预提结果表 aop_dcs_ap_request
中数据都是未上传状态，即 UPLOAD_FLAG 为 null 、 0 或者 3 ）
1. 根据时间戳清除表 aop_dcs_ap_request 中数据

```sql
delete from AOP_DCS_AP_REQUEST a where a.timestamp = '1517441400';
```

2. 代⼊时间戳，重新执⾏以下存储过程
PKG_FUEL_FEE_ACCRUED.FUEL_FEE_CALCULATION
PKG_FUEL_FEE_ACCRUED.FUEL_FEE_EXPRESS
PKG_FUEL_FEE_ACCRUED.FUEL_FEE_OTHER
PKG_FUEL_FEE_ACCRUED.FUEL_FEE_FOR_SOA
数据核对

```sql
-- 如果需要彻底的重新执⾏
BEGIN
PKG_FUEL_FEE_ACCRUED.FUEL_FEE_MAIN_JOB('N');
END;
/
```

功能：航油费 - 财务共享 - 共享预提查询
根据导出的结果表，⽐对结果表中加油数量、汇率、油价、币种是否正确
检查 aop_dcs_ap_request 中⾦额与共享预提查询中是否⼀致，检查 aop_dcs_ap_request 中机型、航线、币种是否为空
